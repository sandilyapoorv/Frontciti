# /frontciti distill

## Purpose

Strip a design to its essence by removing redundant elements, visual noise, and unnecessary complexity.

## Use When

Use when a UI feels cluttered, over-explained, over-designed, or burdened by too many choices.

## Inputs To Inspect

Primary user goal, actions, navigation, copy, component nesting, visual decoration, information architecture, and repeated content.

## Workflow

1. Identify the one primary outcome.
2. Separate essential elements from optional or redundant elements.
3. Remove, merge, hide, or progressively disclose secondary content.
4. Reduce visual variety while preserving hierarchy.
5. Simplify code and component structure where possible.

## Output Expectations

The interface should become easier to understand and faster to use without losing necessary function.

## Anti-Patterns

- Removing information users need.
- Making everything small and low contrast.
- Confusing mystery with minimalism.
- Simplifying accessibility out of the UI.

## Acceptance Checklist

- Cognitive load is lower.
- Primary action is clearer.
- Redundant elements are removed.
- Necessary paths remain available.
