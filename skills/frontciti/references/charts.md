# Charts And Data Visualization

Charts must support decisions.

## Rules

- Give every chart a title or clear context.
- Show axes labels where relevant.
- Make units visible.
- Make legends understandable.
- Include empty states.
- Avoid chart types that obscure comparison.
- Use tables when exact values matter.
- Use charts when pattern recognition matters.

## Common Choices

- Line chart: trend over time.
- Bar chart: category comparison.
- Stacked bar: composition by category.
- Pie or donut: simple share of whole with few categories.
- Area chart: cumulative trend.
- Heatmap: density or intensity.
- Funnel: conversion steps.
