# Frontciti Design Philosophy

Good frontend design is product understanding made visible.

## Principles

1. Fit the interface to the product's actual subject.
2. Make the hero or primary screen express the thesis.
3. Use typography to create personality before adding decoration.
4. Let layout guide attention and sequence.
5. Use color for mood, hierarchy, state, and meaning.
6. Use spacing to create rhythm and confidence.
7. Use motion to reveal relationships or guide attention.
8. Make product UI optimize task completion.
9. Make marketing UI build trust and conversion.
10. Give each design at least one memorable idea.
