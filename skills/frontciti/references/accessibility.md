# Accessibility

Accessibility is mandatory.

## Checklist

- Use semantic HTML.
- Buttons are buttons.
- Links are links.
- Inputs have labels.
- Form errors are connected to fields.
- Focus states are visible.
- Keyboard navigation works.
- Color contrast is sufficient.
- Motion respects reduced motion.
- Images have useful alt text or are marked decorative.
- Modals trap focus and restore focus.
- Menus are keyboard usable.
