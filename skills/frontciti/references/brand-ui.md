# Brand And Marketing UI

Marketing UI must explain, persuade, and build trust.

## Requirements

- Clear audience.
- Clear problem.
- Clear solution.
- Clear proof.
- Clear CTA.
- Strong above-the-fold message.
- Specific copy.
- Trust-building visuals.
- Good mobile reading rhythm.

## Section Flow

Common flow:

1. Hero.
2. Problem.
3. Product explanation.
4. How it works.
5. Proof and features.
6. Use cases.
7. Pricing.
8. FAQ.
9. Final CTA.

Use only sections that serve the page.
