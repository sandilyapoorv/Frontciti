# Layout

Space is a design material.

## Principles

- Group related elements tightly.
- Separate distinct groups generously.
- Let hierarchy emerge from size, weight, position, contrast, and space.
- Use cards only when grouping or action boundaries need a container.
- Use Flexbox for one-dimensional layout and Grid for two-dimensional layout.
- Use container queries for reusable components when available.

## Rhythm

Avoid equal spacing everywhere. Strong pages alternate:

- Tight content groups.
- Generous section breaks.
- Dense evidence.
- Open breathing room.
- Large focal moments.
- Quieter supporting areas.

## Density

Marketing and brand pages can use generous spacing. Product UI should use predictable density. Dashboards can be dense when users need comparison, but density needs grouping and scan paths.

## Responsive Structure

Responsive design is structural:

- Navigation collapses deliberately.
- Tables get cards, horizontal scroll, priority columns, or drill-down.
- Charts simplify or become summaries.
- Modals can become sheets.
- Touch targets remain usable.

## Avoid

- One layout family repeated across every section.
- Three identical feature cards as the default.
- Nested cards.
- Complex flex percentage math.
- Arbitrary spacing values.
- Mobile layouts that only shrink desktop.
