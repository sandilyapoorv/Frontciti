# Product UI

Product UI is different from marketing UI.

## Priorities

1. Help users complete tasks.
2. Show system state clearly.
3. Make next actions obvious.
4. Reduce cognitive load.
5. Support realistic data density.
6. Reveal complexity progressively.
7. Protect users from destructive mistakes.

## SaaS Screens

- Dashboard.
- List and table views.
- Detail pages.
- Create and edit forms.
- Settings.
- Billing.
- Onboarding.
- Notifications.
- Reports.
- Client portals.

## Checklist

- Navigation is clear.
- Data is grouped logically.
- Filters, search, and actions exist where needed.
- Empty states explain what to do.
- Error states explain how to recover.
- Statuses are visible.
- Permissions and account context are clear.
