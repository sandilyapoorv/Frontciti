# Forms

Forms must be clear, forgiving, and easy to complete.

## Rules

- Use labels, not placeholder-only fields.
- Add helper text where needed.
- Validate at useful moments.
- Explain how to fix each error.
- Make required fields clear.
- Group related fields.
- Keep destructive actions separate.
- Show success state after submission.
- Preserve entered data after errors.
- Use suitable input types.
