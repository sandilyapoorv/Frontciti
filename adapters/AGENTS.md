# Frontciti Agent Instructions

Use the Frontciti skill for frontend design, redesign, SaaS UI, dashboards, landing pages, components, critique, polish, hardening, responsive behavior, accessibility, UX copy, and GSAP motion.

Skill location in this repository:

`frontciti-skill/skills/frontciti/SKILL.md`

When a frontend request matches this scope:

1. Read `frontciti-skill/skills/frontciti/SKILL.md`.
2. Follow its command router when the user names a `/frontciti` command.
3. Otherwise infer the closest mode, then read only the relevant command and reference files named by `SKILL.md`.
4. Preserve the existing stack and project conventions.
5. Before finishing, run the Frontciti final self-audit and verify responsive behavior, accessibility, states, and motion.

Do not bulk-load every Frontciti reference by default. Use progressive disclosure.
